## Notes:
Đa số chúng ta chỉ làm độc giả, khán giả mà không viết sách, trình diễn. Như thế chúng ta đơn thuần chứng kiến, bàng quan. Ít nhât ta cũng cần tham dự vào chính những sự kiện ấy
